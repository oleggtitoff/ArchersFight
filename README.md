# ArchersFight
Little SFML game

Educational project

I used MS Visual Studio 2017 v15.9.5 & SFML-2.5.1

SFML - Simple and Fast Multimedia Library
https://www.sfml-dev.org/

The project reveals some of the library's capabilities.
There is work with graphics and audio.
